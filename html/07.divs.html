Styling in Divs (Class / Id / Inline)

Class-Styled Div

This box is styled using seperate style class. Till now we are using inline style sheet but this time we use a style class to decorate the div. The class style is useful when we want a unified style for the entire div. If we want to style any element inside the div seperately, the inline style still there.

Now one may ask - if the class and inline style conflicts with each other then what will happen? The answer is simple - the inline style always has the higher presedence.

Id-Styled Div

This box is styled using seperate style id. Id is a unique identifier, so an id should use only once in a html file. For example if we use an id identifier id="div1-header" to an element we should not use this id again in the same html file.

On the contary, class is a non-unique identifier. The same class="button" can be used on multiple items within a single document.
