Semantic HTML

Semantic HTML is the use of HTML markup to reinforce the semantics, or meaning, of the content in webpages and web applications rather than merely to define its presentation or look.

Common Tags:

  • header: Heading of the whole page.
  • footer: Foot note related to the page or author.
  • nav: Navigation window to go different section or page.
  • main: Main content of the page.
  • article: An independent section of content.
  • section: A thematic grouping of content, typically with a heading.
  • aside: Content related to the main content, like sidebars.
  • div: Divide page into small divisions.
  • table: Create tables.
  • lists: Ordered, unordered and description lists.
  • figure: Self-contained content, like images with captions.
  • figcaption: Caption for the figure element.
